# 📊 Excel Project - Data Analysis

## 📌 Project Overview
This project is created using Microsoft Excel to perform data analysis, calculations, and visualization. It includes structured data, formulas, and formatted tables to generate meaningful insights.

---

## 🧾 Features
- Data entry and organization
- Use of formulas (SUM, AVERAGE, IF, etc.)
- Relative and Absolute cell referencing
- Summary sheet creation
- Conditional formatting
- Charts and graphs for visualization

---

## 📁 File Structure
- `PR1.xlsx` → Main Excel file containing all data and analysis
- `Summary_Sheet` → Displays calculated results and summary

---

## ⚙️ How to Use
1. Open the file `PR1.xlsx` in Microsoft Excel.
2. Navigate through different sheets.
3. Enter or modify data as required.
4. View automatic calculations in the summary section.

---

## 🧮 Formulas Used
- `SUM()` → To calculate total
- `AVERAGE()` → To find mean values
- `IF()` → Logical conditions
- Absolute Reference (`$A$1`)
- Relative Reference (`A1`)

- Proper alignment and borders
