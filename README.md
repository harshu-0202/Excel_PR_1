📊 Excel Project – Student Summary Sheet
📌 Project Overview

This project is created using Microsoft Excel to manage and analyze student marks efficiently. It includes calculations like total marks, bonus addition, and formatted tables for better readability.

🎯 Objectives
Create a structured summary of student marks
Use formulas and references (relative & absolute)
Apply formatting for a clean and professional look
Automate calculations like totals and bonus
🧾 Features
📋 Student Data Table
Columns: Student Name, Marks1, Marks2, Total, Bonus
➕ Automatic Total Calculation
🎁 Bonus Calculation using Absolute Reference
🎨 Formatted Table
Bold headers
Colored cells for better visibility
🛠️ Tools Used
Microsoft Excel
Basic Excel formulas:
=SUM()
Relative Reference (e.g., A1, B1)
Absolute Reference (e.g., $A$1)
📂 File Structure
Project 1.xlsx → Main Excel file containing:
Summary Sheet
Student data
Calculations and formatting
▶️ How to Use
Open the Excel file (Project 1.xlsx)
Go to Summary_Sheet
Enter student names and marks
Total and bonus will calculate automatically
Modify values if needed
